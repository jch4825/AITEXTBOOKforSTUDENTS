# m6-l1 비주얼 사회적 상황이야기 이미지 프롬프트

공통: `yoona-sheet.png`, `aimi-sheet.png`를 참조하며 같은 마트와 교복을 유지합니다. 청소년용 저채도 애니메이션 16:9이며 글자·숫자·브랜드·말풍선·UI·로고·워터마크는 없습니다.

1. 마트에 가기 전 윤아가 샌드위치 목적·사람 수·예산·집의 재료 그림 카드를 정리합니다.
2. 아이미가 핵심 재료와 여러 간식이 섞인 넓은 장보기 목록을 제안합니다.
3. 마트에서 목록, 집에 있는 치즈, 가격, 예산, 안전 정보를 비교합니다.
4. 필요한 빵과 채소만 담고 수정한 그림 목록과 장바구니를 확인합니다.
