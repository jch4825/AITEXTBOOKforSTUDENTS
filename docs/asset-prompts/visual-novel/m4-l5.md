# m4-l5 비주얼 사회적 상황이야기 이미지 프롬프트

공통: `yoona-sheet.png`, `aimi-sheet.png`, 마지막 장면은 `minjun-sheet.png`를 참조합니다. 같은 방과 후 교실과 사진을 유지합니다. 청소년 교육용 저채도 애니메이션 16:9이며 실제 개인정보, 읽을 글자·숫자·말풍선·UI·로고·워터마크는 없습니다.

## 장면 1 — 사진 부탁 받기
윤아가 태블릿의 익명 인물 아이콘과 사진 요청 그림을 보고 전송하지 않은 채 멈춰 생각합니다. 상대의 정체는 드러나지 않습니다.

## 장면 2 — 보낼 사진 고르기
윤아가 교실 활동 사진을 선택하려 합니다. 사진에는 친구 얼굴, 교복 이름표 모양, 창밖 학교 위치 단서가 작게 함께 보입니다.

## 장면 3 — 개인정보 단서 찾기
윤아가 사진을 확대해 이름표 영역, 친구 얼굴, 창밖 표지 모양을 각각 경고 테두리로 표시합니다. 실제 글자는 없습니다.

## 장면 4 — 가리고 안전하게 공유하기
윤아와 민준 선생님이 이름과 얼굴과 위치 단서를 흐리게 가린 사진을 확인합니다. 제한된 학교 과제 공간을 자물쇠와 교실 그림 기호로 나타냅니다.
