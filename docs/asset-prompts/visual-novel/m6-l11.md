# m6-l11 비주얼 사회적 상황이야기 이미지 프롬프트

공통: `yoona-sheet.png`, `aimi-sheet.png`를 참조하며 같은 학교 동아리 교실과 자기소개 카드 스타일을 유지합니다. 청소년용 저채도 애니메이션 16:9이며 실제 개인정보, 글자·숫자·말풍선·UI·로고·워터마크는 없습니다.

1. 윤아가 그림 그리기·식물 돌보기·함께 할 활동을 담은 자기소개 초안을 만듭니다.
2. 아이미가 인사·흥미·강점·함께 할 일 순서와 짧게 고치기, 개인정보 최소화를 제안합니다.
3. 윤아가 취미와 강점은 남기고 집 위치 단서는 빼며 자신의 말투로 순서를 다시 놓습니다.
4. 윤아가 수정한 소개 그림 카드를 학교 동아리 친구들에게 보여 주며 자연스럽게 소개합니다.
