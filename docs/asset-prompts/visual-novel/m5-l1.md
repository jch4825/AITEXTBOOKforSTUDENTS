# m5-l1 비주얼 사회적 상황이야기 이미지 프롬프트

공통: `yoona-sheet.png`, `aimi-sheet.png`를 참조합니다. 낮의 같은 한국 버스 정류장과 교복을 유지합니다. 청소년용 저채도 애니메이션 16:9이며 글자·숫자·말풍선·UI·로고·워터마크는 없습니다.

1. 윤아가 안전한 정류장에서 도서관 약속 그림과 버스 도착 기호를 확인합니다.
2. 예정 시간이 지났지만 버스가 오지 않아 윤아가 빈 도로를 바라봅니다.
3. 늦어진 버스 기호, 가까워진 약속 시간, 연락 수단 카드를 함께 비교합니다.
4. 윤아가 보호자에게 연락하며 밝은 정류장에서 기다리기와 안전한 도움받기를 비교합니다.
