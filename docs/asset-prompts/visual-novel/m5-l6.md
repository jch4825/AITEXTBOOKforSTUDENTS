# m5-l6 비주얼 사회적 상황이야기 이미지 프롬프트

공통: `jinwoo-sheet.png`, `aimi-sheet.png`, 마지막은 `minjun-sheet.png`를 참조합니다. 같은 학교 앞 환경을 유지합니다. 청소년용 저채도 애니메이션 16:9이며 실제 주소, 글자·숫자·말풍선·UI·로고·워터마크는 없습니다.

1. 진우가 학교 앞에서 중앙도서관 길을 아이미에게 묻습니다.
2. 아이미가 멀리 있는 현대식 다른 도서관 지점을 지도 그림으로 보여 줍니다.
3. 진우가 학교 출발점, 공원, 벽돌 도서관 그림 단서를 더해 다시 말합니다.
4. 진우와 민준 선생님이 공원 옆 벽돌 도서관과 버스 노선을 공식 안내로 확인합니다.
