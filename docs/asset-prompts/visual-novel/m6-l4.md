# m6-l4 비주얼 사회적 상황이야기 이미지 프롬프트

공통: `jinwoo-sheet.png`, `aimi-sheet.png`, 마지막은 `minjun-sheet.png`를 참조합니다. 같은 밝은 버스 정류장을 유지합니다. 청소년용 저채도 애니메이션 16:9이며 실제 노선, 글자·숫자·말풍선·UI·로고·워터마크는 없습니다.

1. 진우가 앱으로 정류장·방향·버스 도착 그림 정보를 확인합니다.
2. 앱은 곧 도착을 표시하지만 도로에는 버스가 없습니다.
3. 정류장 지연 공지, 공식 운행 그림, 안전요원과 주변 밝기를 함께 확인합니다.
4. 보호자에게 알리고 다음 버스 대기와 확인된 지하철 대안을 안전 기준으로 비교합니다.
