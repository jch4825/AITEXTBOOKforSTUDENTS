## 사용법 (Gemini)

1. **캐릭터 시트 이미지를 함께 첨부한다.** 프롬프트만으로는 캐릭터가 매번 달라진다.
   캐릭터 설정 자료집 4종은 `docs/character-sheets/`에 있다
   (`jinwoo-sheet.png` · `yoona-sheet.png` · `minjun-sheet.png` · `aimi-sheet.png`).
   이를 첨부하고 *"첨부한 캐릭터 시트의 디자인을 그대로 유지해서"* 라고 지시하는 것이 일관성의 핵심.
   (시트 안의 한글 라벨은 생성 AI 특성상 오탈자가 있으나, 시각 참조용이므로 무방.)
2. 아래 **[공통 블록]** 을 복사해 프롬프트 맨 앞에 붙인다.
3. 각 차시의 이미지/영상 프롬프트를 그 뒤에 이어 붙인다.
4. 결과물 저장 규격:
   - 이미지: `public/lessons/{차시ID}.png` — 16:9 (예: `m1-l1.png`)
   - 영상: `public/lessons/{차시ID}.mp4` — 16:9, 8초 내외
   - 앱 적용은 디자인 로드맵 D2(무대 컴포넌트)에서 일괄 연결 예정.

## ⚠️ 불변 규칙 (모든 생성에 적용)

- **말풍선은 반드시 속이 빈 상태로** — 글자·문장을 절대 넣지 않는다 (생성 AI의 한글 뭉개짐 방지, 텍스트는 앱이 표시).
- 화면 안에 **글자·숫자·로고·워터마크 금지**.
- **저자극 원칙**: 점멸·번쩍임·빠른 컷·흔들리는 카메라 금지. 채도는 낮게, 파스텔.
- 인물은 항상 긍정적이거나 차분한 표정 (모듈 4의 "위험 상황"도 무섭지 않게).

## [공통 블록] — 이미지용 (프롬프트 맨 앞에 복사)

```
따뜻한 그림책풍 플랫 일러스트. 부드러운 파스텔 톤(낮은 채도), 깨끗하고 단순한 배경,
부드러운 자연광, 16:9 비율. 화면에 글자·숫자·로고를 넣지 말 것.
말풍선을 그릴 경우 반드시 아무 글자도 없는 빈 흰색 말풍선으로 그릴 것.

등장 캐릭터 (첨부한 캐릭터 시트의 디자인을 그대로 유지):
- 아이미: 키 35cm의 작고 귀여운 호버링 로봇. 연분홍빛 흰색의 둥근 몸체, 남색 LED 얼굴
  화면에 하늘색으로 빛나는 눈과 미소, 머리 위 안테나 끝에 파란 구슬, 짧고 둥근 팔,
  바닥에서 살짝 떠 있음.
- 진우: 17세 한국 남학생. 짧고 부스스한 갈색 머리, 남색 블레이저 교복에 빨간 넥타이,
  밝고 활달한 표정.
- 윤아: 17세 한국 여학생. 단정한 갈색 단발머리(일자 앞머리), 남색 블레이저에 베이지
  조끼와 빨간 리본, 차분하고 섬세한 미소.
- 민준쌤: 37세 한국 남자 교사. 안경, 정갈한 검은 머리, 남색 정장에 갈색 넥타이,
  온화하고 차분한 표정.
```

## [공통 블록] — 영상용 (프롬프트 맨 앞에 복사)

```
따뜻한 그림책풍 2D 애니메이션, 부드러운 파스텔 톤(낮은 채도), 16:9, 8초.
카메라는 고정 또는 아주 느린 줌 인만 사용. 흔들림·점멸·빠른 컷·급격한 움직임 금지.
등장인물의 입모임(대사) 없음, 화면 내 글자·자막 없음.
말풍선이 등장할 경우 반드시 아무 글자도 없는 빈 말풍선으로.
분위기는 잔잔하고 따뜻하게.

(캐릭터 묘사는 이미지용 공통 블록과 동일 — 첨부한 캐릭터 시트의 디자인을 그대로 유지)
```
