# 활동(card-pick·matching·sequence) 아이콘 — 미보유 목록

- 목적: 카드 선택·짝맞추기·순서 배열 선택지에 붙일 그림 아이콘. 글자를 못 읽는 학생도 그림만 보고 참여할 수 있게 하기 위함.
- 스타일: 기존 PECS 심볼(`public/lessons/pecs/*.webp`)과 시각적으로 통일감 있게 — 정사각 캔버스, 투명 배경, 단순하고 굵은 형태, 배경색 있는 타일 위에 올라가도 잘 보이는 고대비, 텍스트 없이 그림 하나로 뜻이 전달되는 것. 세부 디테일 최소화(작은 크기에서도 알아볼 수 있어야 함).
- 파일명 규칙: `public/lessons/pecs/{concept}.webp` 에 추가할 예정 (기존 40종과 같은 폴더/네이밍 컨벤션).
- 조사 범위: `src/data/lessons/m1.ts` ~ `m6.ts` 전체 68차시의 `card-pick`(선택지) · `matching`(짝의 좌/우 모두) · `sequence`(항목) 데이터만 추출. `ox`(O/X는 이미 도형으로 처리), `text`/`sim-ai`/`real-ai`(서술·대화 스텝)는 제외.
- 원본 문자열 389개를 추출 → 실제 그림으로 구분되는 개념 약 83개로 정리(그 중 29개는 기존 PECS로 이미 커버됨, 54개가 아래 미보유 목록). 나머지는 아래 "제외한 것" 참고.

## 제외한 것 (의도적으로 목록에서 뺀 항목)

- **모듈 2 전반의 프롬프트 비교 선택지** — "강아지 이름 세 개 추천해줘" vs "음..." vs "뭐" 같은, 문장의 좋고 나쁨을 비교하는 선택지들. 그림 하나로는 "이게 더 좋은 질문이다"라는 판단 자체를 표현할 수 없어서 아이콘화 대상에서 뺐다(글자 그대로 비교해야 학습이 성립하는 내용).
- **의미 없는 오답 채우기용 선택지** — "음...", "아무 말 안 하기", "가만히 있기", "화면 두드리기", "화면을 꺼요" 등 "아무것도 안 함/우물거림"을 나타내는 필러 오답들. 그림으로 구분할 실익이 낮아 제외.
- **반의어/유의어 짝맞추기** — "크다"/"작다", "기쁘다"/"즐겁다" 같은 국어 어휘 짝. 개별 사물이 아니라 낱말 관계를 배우는 내용이라 제외.
- **돈 계산 선택지** — "300원", "2,000원" 처럼 숫자만 다른 정답/오답. 숫자 자체를 그림으로 구분할 수 없어 제외.

## 이미 있어서 재사용 가능 (참고용)

card-pick·matching·sequence 선택지에 실제로 등장했고, 이미 PECS로 커버되는 대표 예시:

- 고양이(`cat`), 코끼리(`elephant`), 토끼(`rabbit`), 거북이(`turtle`), 치타(`cheetah`), 독수리(`eagle`), 공룡(`dinosaur`) — 동물 선택지 다수
- 우산(`umbrella`), 구름(`cloud`), 부채(`hand_fan`), 튜브(`tube`), 선글라스(`sunglasses`) — m6-l5 날씨/옷차림 선택지
- 사과(`apple`), 빵(`bread`), 치즈(`cheese`), 햄(`ham`), 상추(`lettuce`), 냄비(`pot`) — 요리·장보기 관련
- 책(`book`), 연필(`pencil`), 지우개(`eraser`) — 학용품
- 스마트폰(`smartphone`), 음성비서(`ai_speaker`), 전등 스위치(`light_switch`), 선풍기(`fan`) — m1-l2 기계 vs AI 구별
- 옷(`clothes`), 종이컵(`paper_cup`), 간식(`snack`), 물/수도꼭지(`faucet`)
