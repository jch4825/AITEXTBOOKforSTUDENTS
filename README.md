# AI 교과서 — 발달장애 학생을 위한 AI 학습

발달장애 학생이 인공지능을 안전하고 즐겁게 배울 수 있도록 만든 온라인 교과서입니다.

## 모듈

1. **AI가 뭐야?** — 개념 이해
2. **AI랑 말해보기** — 대화와 프롬프트
3. **AI랑 같이 배우기** — 학습 활용
4. **AI 안전하게 쓰기** — 안전·윤리·자기보호
5. **AI로 문제해결하기** — 논리·사고력
6. **AI랑 일상생활** — 직업·생활 실전

## 시작

```bash
npm install
npm run dev
```

`http://localhost:3000` 에서 확인.
