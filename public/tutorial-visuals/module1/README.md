Module 1 tutorial visuals.

Files in this directory are shown inside the tutorial lesson viewer.
Naming convention:

- `l1-1-visual.png`
- `l1-2-visual.png`
- `l1-3-visual.png`
- `l1-4-visual.png`
- `l1-5-visual.png`

Add future module assets under sibling directories such as:

- `public/tutorial-visuals/module2`
- `public/tutorial-visuals/module3`
