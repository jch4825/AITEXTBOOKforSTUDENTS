# 개별화 교육계획 작성(22 교육과정 반영)

근거: 개별화교육계획 운영 가이드북 (국립특수교육원, 2019)
대상: 특수교육 기본교육과정 1~12학년 / 성취기준 691개 / 교과 16개 / 학년군 5개

## 데이터

파일: basicCurriculumStandards.csv (스킬 폴더 내 동봉)

CSV 컬럼: code / subject / gradeGroup / title / 상 / 중 / 하

```python
import csv
standards = []
with open('basicCurriculumStandards.csv', 'r', encoding='utf-8') as f:
    for row in csv.DictReader(f):
        standards.append(row)
```

학년군: 1~2학년 / 3~4학년 / 5~6학년 / 7~9학년 / 10~12학년

검색 예시:
- 코드: [s for s in standards if s['code'] == '[2건생01-01]']
- 교과+학년군: [s for s in standards if s['subject']=='국어' and s['gradeGroup']=='1~2학년']
- 키워드: [s for s in standards if '의사소통' in s['title']]

## 성취수준 해석

- 상: 독립 수행, 일반화 가능
- 중: 언어적 촉구, 모델링 등 부분 지원으로 수행
- 하: 신체적 지원, 보조기구, AAC 등 최대 지원으로 수행

## IEP 작성 절차

IEP는 학기계획 / 학기평가 / 월계획 / 월평가로 구분하여 운영한다. 단계별로 질문하고 답변을 받은 후 다음 단계로 진행한다.

### STEP 1. IEP 유형 확인

"생활지원 중심인가요, 교과 중심인가요?" 를 가장 먼저 질문한다.

- 생활지원: 성취기준 검색 및 [관련 성취기준] / [대체 성취기준] 출력 생략. 목표 분석 후 평가방법, 평가초점으로 진행.
- 교과: 아래 전체 절차 진행.

### STEP 2. 기본 정보 수집

다음 세 가지를 한 번에 질문한다. 과목은 절대 묻지 않는다.

1. 학생의 학년
2. 학생의 현행 수준 (출력 시 ~~함. 형태로 정리)
3. 학기 목표 (교사 작성 또는 성취기준 탐색 후 도출)

### STEP 3. 학기 목표 분석

3-1. 맞춤법, 문맥 점검 후 ~한다. 형태로 정리. 수정 시 원문과 수정문을 함께 제시하고 확인을 구한다.

3-2. [관련 성취기준] - basicCurriculumStandards.csv에서 동일 학년군 내 관련 성취기준 최대 3개 제시. [기본교육과정 내 대체 성취기준] - 동일 학년군보다 타 학년군이 더 적합하면 동일 학년군을 먼저 제시 후 타 학년군을 추가로 최대 3개 제시. 성취기준 출력 시 코드 뒤에 과목명을 절대 붙이지 않는다. 형식: [코드]성취기준 문장

3-3. *평가방법* - 목표에 가장 적합한 평가 방법 제시.

3-4. *평가초점* - 평가에 필요한 활동이나 수행을 ~~가? 형태로 기술.

### STEP 4. 월별 계획

"이 학기는 몇 월부터 몇 월까지인가요?" 를 질문한다. 한 달 단위, 최대 3달로 구분하여 제안한다. 3달 초과 시 기초 - 심화 - 일반화 단계로 설계한다.

각 월별 출력:
- 월별 목표: ~한다.
- 교육내용: ~~하기
- 교육방법: ~한다.
- 평가방법: ... / 평가준거: ~~하는가?

## IEP 결과물 출력 형식

```
[IEP 학기계획]
- 학년: ○학년
- 현행 수준: ~~함.
- 학기 목표: ~한다.
* [관련 성취기준]
  - [코드]성취기준 문장
[기본교육과정 내 대체 성취기준] (해당 시)
  - [코드]성취기준 문장 (학년군)
*평가방법* ...
*평가초점* ...가?
---
[월별 계획]
[○월 계획]
- 월별 목표: ~한다.
- 교육내용: ~~하기
- 교육방법: ~한다.
- 평가방법: ... / 평가준거: ~~하는가?
```

## IEP 평가 작성

학기평가, 월평가 요청 시 성취수준(상/중/하)으로 표시하고 ~~함. 형태로 기술. 미달성 목표는 이월 여부를 안내한다.

## 출력 형식 규칙 (모든 결과물 공통)

- 특수문자: [ ] ( ) , . * - / + 만 허용. 그 외 모든 기호 사용 금지.
- ~하기로 끝나는 문장: 줄 바꿈 허용.
- 마침표로 끝나는 문장: 새 항목, 새 문단 시작 시에만 줄 바꿈 허용.
- 문장 중간 줄 바꿈: 절대 금지.

## 주의사항

- 코드 형식: [2건생01-01] (대괄호 포함)
- 성취수준 용어: 상/중/하 는 A/B/C 와 동일한 의미이며 혼용 가능하다.
- 이 데이터는 기본교육과정(특수교육 전용)이며 일반 교육과정과 다르다.
