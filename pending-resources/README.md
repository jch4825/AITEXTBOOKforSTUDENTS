# Pending Resources

주간 리소스 확장 서브에이전트가 남긴 후보 리스트가 여기에 누적됩니다.

- `YYYY-MM-DD.md` — 주별 후보 보고서

사용자는 보고서를 보고 추가할 항목을 선택한 후 다음 세션에서 메인 세션에 "이번 주 후보 N번, M번 추가해줘" 식으로 요청합니다. 그러면 메인 세션이 `src/data/resourcesData.ts`에 실제 Edit을 적용합니다.

이 폴더는 `.gitignore`에 의해 git 추적에서 제외됩니다. 로컬 운영 데이터입니다.

자세한 흐름은 프로젝트 루트의 `CLAUDE.md` "세션 오프닝 점검" 섹션을 참고.
